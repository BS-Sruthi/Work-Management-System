Common Features:

1. Employee registration
2. Login and logout system
3. Different dashboards for Manager and Employee
4. Edit employee and manager details
5. Task notifications

Manager Module:

1. View all employees
2. View employee details
3. Assign tasks to employees
4. Edit or reassign tasks
5. Check employee work status
6. View pending and completed reports
7. Track task status:
   * To Do
   * In Progress
   * Completed

Employee Module:

1. View profile details
2. View assigned tasks
3. Update task progress
4. Mark task as completed
5. View task deadlines
6. Receive manager notifications
7. View completed task history

Task Management:

1. Create tasks
2. Update task details
3. Reassign tasks
4. Track task status
5. Monitor task progress
6. Generate simple work reports
